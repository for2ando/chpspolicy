[これは何?]
Windows PowerShell を初めて実行する際に必要となる実行ポリシーの変更を半自動化するためのバッチファイルです。PowerShellをふくむアプリケーション等を配布する場合に、アプリケーションのインストールスクリプト等から呼び出して使うことを主に想定しています(もちろん単体で使うこともできます)。

[使い方]
実行ポリシーを変更したいWindowsマシンで、次のバッチファイルのいずれかを「管理者として実行」から実行して下さい。
chPsRemoteSigned.bat	実行ポリシーを RemoteSigned に変更します。
chPsRestricted.bat	実行ポリシーを Restricted に変更します。
chPsUnrestricted.bat	実行ポリシーを Unrestricted に変更します。
インストールスクリプト等から呼び出す場合は、インストールスクリプト自体を「管理者として実行」する必要があるでしょう。

変更後には、その時点の実行ポリシーを表示します。
